# 🎲 Dice Roller Game

A simple command-line dice rolling game built in Python.

## What it does
Roll up to 100 dice at once and see the results printed instantly. Keep rolling as many times as you want, or type `n` to quit.

## Example
Do you want to roll a dice? (y/n): y
How many dices do want to roll? (1-100): 3
[4, 1, 6]
Do you want to roll a dice? (y/n): n
Thanks for using our app!

## What I learned
This was my first Python project — I started with an if/elif chain for each dice count, then refactored it into a list comprehension to handle any number of dice cleanly.

# 🎲🔢 First Python Projects

A small collection of my early Python practice projects — simple command-line games built while learning the basics.

## Projects

**Dice Roller (`dice_roller.py`)** — Roll up to 100 dice at once and see the results printed instantly. Keep rolling as many times as you want, or type `n` to quit.

Run it: `python dice_roller.py`

Example:
Do you want to roll a dice? (y/n): y
How many dices do want to roll? (1-100): 3
[4, 1, 6]
Do you want to roll a dice? (y/n): n
Thanks for using our app!

**Number Guessing Game (`guess_number.py`)** — Guess a random number between 1-100. The game tells you whether to guess higher or lower until you get it right.

Run it: `python guess_number.py`

Example:
Guess the number between 1-100: 50
Guess lower!
Guess the number between 1-100: 25
Guess higher!
Guess the number between 1-100: 37
Congratulations, You got it!

## What I learned

These were my first Python projects. I started with repetitive if/elif chains and gradually refactored toward cleaner solutions — like replacing multiple `random.randint()` calls with a single list comprehension.
